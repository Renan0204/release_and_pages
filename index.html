GitHub Pages • Deploy automático

Site simples, rápido e elegante.

Esta página é um template minimalista pronto para publicar no GitHub Pages. Edite o conteúdo, suba para a branch main e o deploy acontece sozinho.

Responsivo

Layout que se adapta bem do celular ao desktop sem esforço.

Sem dependências

HTML + CSS puro. Rápido para carregar, fácil de manter.

Dark/Light

Respeita o tema do sistema com prefers-color-scheme.

Pronto pro Pages

Combine com um workflow de deploy “on push”.

Como publicar

  1. Coloque este arquivo como index.html na raiz do repositório.
  2. Faça commit e push na branch main.
  3. Nas configurações do repositório, ative Settings → Pages → Source: GitHub Actions.

Dica: personalize cores alterando as variáveis CSS no início do arquivo.
